Install Instructions:

If you run into any problems during or after the installation please join our Discord and read the FAQ, if you still run into issues feel free to use the support channel

Before installing the collection:

  • A nexus premium account is highly recommended for an automated installation. Otherwise, the installation will take a lot of manual clicking. (otherwise you need to manually download over 1500 mods)
  • Uninstall Skyrim SE and remove the following 2 folders: 
    "Steam\steamapps\common\Skyrim Special Edition"
    "Documents\My Games\Skyrim Special Edition" (optionally backup your "Saves" folder)
  • If you installed any other Skyrim SE collections with Vortex: Close Vortex, press "windows + R", enter "%appdata%/Vortex/skyrimse", and delete the "userlist" file
  • You need a fresh install of Skyrim SE on Steam in English. (other versions are not compatible)
  • You also need the Anniversary Upgrade DLC (Constellations doesn't work without it!)
  • Start the vanilla game and download the Anniversary Upgrade DLC
  • Head to the Skyrim directory Data folder (Steam\steamapps\common\Skyrim Special Edition\Data) and ensure you have 148 files starting with 'cc' (if you don't have 148, follow these instructions)
  • Install Microsoft Visual C++ Redistributable Package
  • WARNING The following website and the download is currently flagged as insecure, proceed with caution, I'm currently investigating this issue, Head to the ENB website, scroll down and click "v0.499", scroll down again, and click the arrow beside "download"
  • Open the Wrapper Version folder and copy "d3d11.dll" and "d3dcompiler_46e.dll" and paste them into "Steam\steamapps\common\Skyrim Special Edition" folder

Installing the collection:

  • Install Vortex and open it
  • If you're using an old installation of Vortex, ensure that your Vortex is updated to the latest version
  • Open Vortex and head to the "Games" section, search for "Skyrim Special Edition" and ensure that the game is managed and activated
  • Head to the "Settings" section in Vortex, navigate to the "Mods" section and ensure that "Hardlink Deployment" is active as your deployment method. (if Hardlink Deployment is not an option please check out the requirements for Hardlink Deployment)
  • Head to the "Settings" section in Vortex and disable both of these options: "Install Mods when downloaded" and "Run FNIS on Deployment Event"
  • Head to the "Mods" section in Vortex, click any mod, press [Ctrl]+[A], click "Remove" at the bottom, and confirm to remove all old mods. (Repeat this process until no mods are left)
  • Install the collection (Press "Add to Vortex" at the top of this page)
  • When prompted about which profile you want to use, press "Use new Profile" and switch to that profile when asked
  • Don't install all optional mods. Check out the "Optional Mods" section below for more info.

After installing the collection:

  • If there are unresolved file conflicts or other errors: Please refer to the guide posted on our Discord in the #faq channel
  • Head to the "Mods" section and "Deploy Mods" (repeat this process whenever you change ANYTHING in the collection)
  • Head to the "Plugins" section and enable all "Disabled" plugins (press [Ctrl]+[A] to mark all)
  • After that press "Sort Now" and wait until the process is done (repeat this process whenever you change ANYTHING in the collection)
  • Head to the "Dashboard" section and press the 3 dots from the "Skyrim Script Extender", click "Make Primary" (if it's not already)
  • Start the Skyrim Launcher via Vortex and configure your graphic settings (deploy mods after changing settings)
  • Always start the game with the upper left "Play" icon in Vortex (the first launch can take up to 10 minutes)
  • Check out the following sections below: Update Instructions, Starting Guide, Optional Mods, Widescreen Support, Keybinds + Gamepad, Known Issues
  • After starting a new game open the journal [J] and follow the instructions (Check out the Starting Guide below for more information)
  • If you enjoy the collection I'd really appreciate an endorsement :)

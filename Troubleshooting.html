Constellations Troubleshooting

 

Vortex errors

 

1. Failed to install dependency

Click on "more".

If it says that it failed with code 429 this means you have exceeded the nexus daily API request limit, it fully resets at midnight GMT, but some small parts are restored every hour. 

 

5. Missing masters

When you get the error, click on 'more', you will see two lists of plugins we are interested in the right one, those are the missing ones (left list is list of plugins that require the missing ones)

Check which plugins are missing. If any of them begin with “CC” that indicates you are missing the Creation Club content that is included in the AE DLC. To install them:

  1. In Mods, select 'Purge Mods'.,
  2. Verify integrity of game files in Steam,
  3. Launch the game vanilla,
  4. Install the AE DLC in the 'Creations' menu by clicking 'o' and selecting: 'Download all owned creation...',
  5. Head to the Skyrim directory Data folder Steam\steamapps\common\Skyrim Special Edition\Data and ensure you have 148 files starting with cc (if you don't have 148, follow steps 2-4 again or try to download them manually from CC menu in game),
  6. Deploy mods and sort your plugins,

3. If the plugins do not begin with "CC", deploy mods, navigate to Plugins, filter by 'Disabled' and enable any disabled plugins. 

4. If you still have missing masters and not other vortex warnings/errors, post screenshot of the list in #cnst-support channel on discord.

 

 

Missing Masters

Missing masters will cause your game to crash at launch or not load. Ensure you have installed the Anniversary Edition DLC Creation Club content in the Skyrim main menu prior to installing Constellations and that you have enabled all disabled plugins.

 

Mod Conflicts/Missing Mod Rules

Uninstall any mods not included in Constellations and follow the instructions posted below. If you have an excessive number of unresolved mod rules, reinstall the collection according to the instructions below. If any unresolved mod rules remain, post a screenshot of them in ⁠cnst-support

 

Mod Cycles

Mod cycles are usually the result of attempting to resolve Mod Conflicts by yourself. Uninstall any mods not included in Constellations and reinstall the collection.

 

Game Crashes at Startup or Won't Load

This is usually due to missing masters or improperly sorted plugins. See Missing Masters above and ensure you have sorted plugins and waited for the process to finish.

 

Game Crashes Repeatedly During Gameplay

This is usually result of user hardware and/or having your graphics setting set too high.

 

SKSE64 Loader Error

SKSE64 Loader errors are usually from manually installing the wrong version of SKSE.
